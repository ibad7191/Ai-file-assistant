# 🤖 Chatboard AI Assistant

A powerful multimodal AI assistant built with **Flask** and **Google Gemini API**. Upload images, PDFs, audio files, or just chat with the AI. Get responses in clean bullet-point format for easy reading.

![Gemini](https://img.shields.io/badge/Gemini-API-orange)
![License](https://img.shields.io/badge/License-MIT-yellow)

---

## ✨ Features

- 📄 **File Upload & Analysis**
  - 🖼️ Image description (objects, colors, setting)
  - 📑 PDF summarization in bullet points
  - 🎵 Audio transcription with key topics
  - 📎 General file analysis

- 💬 **Smart Chat Interface**
  - Real-time conversation with AI
  - Session management for multiple users
  - Conversation history tracking
  - Clean bullet-point responses

- 🎨 **Modern UI**
  - Responsive design (mobile-friendly)
  - Drag & drop file upload
  - Typing indicators
  - File upload status
